# Скрипт для автоматизированной проверки ответов на вопросы теста

## Общая информация
Все вопросы будут выбираться случайно из общей большой базы данных (размер БД - около 200 вопросов). 
Количество вопросов в одном тесте - около 20. Количество студентов - 100 человек.

## Постановка задачи
Нужно запрограммировать на Python скрипт, который должен по заданному xls-файлу с ответами:
- сформировать список (list) вопросов
- для каждого вопроса найти все ответы студентов на этот вопрос
- в цикле по блокам вопросов (размер блока question_block_size = 10 вопросов) задать вопрос LLM через Polza.ai с
  просьбой оценить ответы всех студентов на эти вопросы по 100-бальной шкале и вернуть json результатов оценки
- сформировать LaTex-файл отчета с результатами теста в формате:
  ФИО
  Вопрос 1: .... 
  Ответ 1: .... 
  Оценка 1: ...
  Вопрос 2: .... 
  Ответ 2: .... 
  Оценка 2: ...
  ...
  ФИО отсортированы в алфавитном порядке.
  
Помимо этого необходимо реализовать возможность передать файл с указаниями по проверке определённых вопросов 
(например, правильные ответы или правила оценивания, если вопрос - специфический)
